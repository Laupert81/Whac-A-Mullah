# Victory Screen Assets

This folder should contain the following assets:

## victory-background.jpg
- **Description**: Dark blue-ish graveyard themed background
- **Format**: JPEG
- **Usage**: Covers the entire victory/game-over screen regardless of orientation
- **Note**: Should work well with both portrait and landscape orientations
- **Design**: Dark, atmospheric graveyard scene with blue-ish tones

## victory-music.mp3
- **Description**: Victorious/triumphant music that plays on the score screen
- **Format**: MP3
- **Duration**: Should be at least 10-15 seconds
- **Note**: Music by K. Kasyanov
- **Behavior**: Starts immediately when victory screen opens, content fades in after 2 seconds

**Required files:**
1. `victory-background.jpg` - The graveyard background image
2. `victory-music.mp3` - The victory music track
3. `iran-flag.png` - The Iranian flag image (transparent PNG)

Add these files to this folder for the victory screen to work properly.
